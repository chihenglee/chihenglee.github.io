㳋·早餐俱樂部算是滿有特色的一間早餐店，
和現在一般的三明治有點不一樣，個人風格強烈。
當天吃的三明治口味有點太重，吐司很厚實，像厚片吐司，
起司雖然很多，但是有點太重鹹了，還好有青蔥和洋蔥稍微中和一些。
吐司烤得酥酥的，不過吃起來有點油，洋蔥味道略嗆，較為搶味。
飲料部分的選擇不多，讓我頓時不太知道要點什麼，
最後選了常見的紅茶，對我來說甜度過高，有點可惜。
不過我很喜歡這裡的用餐環境，小小的舒適且悠閒，
希望之後會提供蛋餅，會再來訪(笑)

CSS background

CSS background-color 的功能用來設定網頁背景顏色，
也可以用來設定網頁元素的背景顏色，例如 DIV 區塊、span 區域、
表格背景甚至是文字背景顏色，都可以使用 background-color 的技巧來設計，
background-color 可以使用的顏色值包含顏色英文名稱、
十六進位制色碼以及 RGB 色碼，
是 CSS 標準的網頁與元素背景顏色設計方式，
所有的主流瀏覽器都支援 background-color 的屬性。
